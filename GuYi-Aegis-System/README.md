# GuYi Aegis Pro - 企业级卡密验证管理系统

基于 PHP + SQLite 的轻量级、高性能卡密分发与验证系统。支持多应用管理、设备指纹绑定、防 Session 劫持等安全特性。

## ✨ 功能特性

- **多租户支持**：单后台管理多个应用（App），独立的 App Key。
- **卡密管理**：支持批量生成（天/周/月/年卡）、导出、解绑、加时、封禁。
- **安全防护**：
  - 基于 HMAC-SHA256 的 Cookie 签名验证。
  - CSRF 防护。
  - 设备指纹绑定（防止一卡多用）。
  - IP 与 UserAgent 异常检测。
- **数据分析**：内置仪表盘，展示活跃设备、库存占比、类型分布图表。
- **轻量级**：无需 MySQL，基于 SQLite，即插即用。

## 🛠️ 安装环境

- PHP >= 7.4
- PDO SQLite 扩展
- Web 服务器 (Nginx/Apache/LiteSpeed)

## 🚀 快速开始

1. **下载源码**
   克隆本项目到网站目录。

2. **配置环境**
   将 `config.sample.php` 重命名为 `config.php`。
   打开并修改 `SYS_SECRET` 为一个随机的复杂字符串（用于加密）。
